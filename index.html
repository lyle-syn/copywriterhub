Maintenance · For whoever's adding the client

Add a
new client

This hub builds itself from one list. Adding a client means copying a template, pasting it in, and swapping the words. No design or code knowledge needed — you're only editing text.

Before you start

Open the hub file in any plain text editor. Search for ADD A NEW CLIENT HERE — there's a ready-made blank template waiting there with instructions right next to it. Keep a backup copy of the file before editing, just in case.

1

Find the data list

Near the top of the file there's a big block that starts with const CLIENTS = [. Every client lives inside it. The presentation code is far below a clearly marked line — you never touch that part.

2

Copy the template

Scroll to ►►► ADD A NEW CLIENT HERE ◄◄◄ at the bottom of the list. Add a comma after the } just above it, then copy the whole template block between the dashed lines.

3

Fill in the real details

Replace every placeholder with the client's real info. Keep each value inside its quotes, and separate list items with commas. The shape looks like this:

id:"client-name", // no spaces, must be unique name:"Full Client Name", short:"Menu name", channels:["Social","Meta Ads"], links:{ combo:"https://docs.google.com/..." }, overview:[ "Point one", "Point two" ], voice:[ "Trait", "Trait" ], // ...and so on, see the template comments
4

Handle the links correctly

If Strategy and Content are the same document, use links:{ combo:"URL" } — the profile will label it "Strategy & Content (same document)".

If they're two separate documents, use links:{ strategy:"URL", content:"URL" }.

If there are no links (like Ishara), use links:{} and they'll show as "Not applicable for this client".

5

When a section doesn't apply

Some clients don't fit the standard mould — that's fine, the template stays identical. For a missing writing breakdown, copy how Engen does it: a {na:true,reason:"..."} block. For a PPC-only client, see Sun Destinations. For a special collaboration workflow, see Ishara's special block.

To put a client in the grouped finance cluster in the menu, add group:"finance". Leave it out otherwise.

6

Save and check

Save the file and open it in a browser. The new client appears in the Clients menu automatically and gets a full profile — same structure as every other client. If the menu looks broken, you most likely missed a comma or a closing quote in step 3. Undo your last change and try again slowly.
